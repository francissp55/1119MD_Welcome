# Check‑Out Instructions

Thank you for staying with us. Before you leave:

- Be sure to make sure the gas cooktop is turned off (yes, it has been left on)
- Empty the dishwasher if needed
- Empty the refrigerator
- Lock all doors and windows
- Return keys

We try to constantly improve amenities. Please let the agent know if there
is anything we can do to improve your comfort

Safe travels!
